# Omid Shahbazi's Reservation Project

### Installation
1. clone the project and change working directory on project folder.
2. create a .env file based on .env.example and place desired values in it.
	```bash
	cp .env.example .env
	```
3. build and run the project
	```bash
	docker compose build && docker compose up
	```
4. go to `http://127.0.0.1:8000/docs` for OpenAPI compatible documentations
